# mega-frontend (plugin)

Frontend design guidance: one skill, `designing-frontends`, for deliberate
visual direction when building or reshaping UI. It grounds the direction in
the brief's subject, plans through a compact token system (palette, type
roles, layout, signature element) before any code, carries a dated
calibration of the default looks AI-generated design currently clusters
around, and treats UX copy as design material.

Verification pairs with `mega-orchestration` (if installed): the
`visual_verify` role gives the rendered result an independent cross-vendor
pass.

## Install

```
/plugin install mega-frontend@megapowers
```
